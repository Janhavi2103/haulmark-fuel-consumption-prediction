# HaulMark Fuel Consumption Prediction

This project was developed as part of the **IIT Madras AI Guild Enrollment Challenge**, where participants were tasked with building machine learning solutions for predicting fuel consumption in mining operations using telemetry and geospatial data.

Our team secured **3rd place institute-wide**, competing against students across all academic years, from first-year undergraduates to final-year students.

The solution combines extensive feature engineering, telemetry analytics, route-based benchmarking, and ensemble machine learning models to predict fuel consumption for mining dump trucks. Multiple approaches including XGBoost, LightGBM, CatBoost, and ensemble methods were evaluated to maximize predictive performance.

Key highlights:

* Telemetry-based fuel consumption prediction
* Geospatial and route-level feature engineering
* Vehicle history and operational pattern analysis
* Ensemble machine learning models
* Comprehensive model evaluation and error analysis

This repository contains the complete training pipeline, feature engineering modules, prediction scripts, and utilities used during the competition.
